Calibration

Precision Calibration waits for the configured electronics warm-up, establishes the force scale and evaluates every static filter preset against the same empty-plate sample. It recommends the fastest preset that reaches the requested noise, but never changes your active selection. The OLED and this panel show the same short instruction at every step.

What will happen 1. Warm-up: if requested, the board waits only for the remaining time since power-up. 2. Empty plate: a fresh tare is taken, then every static filter candidate learns from the same full-rate samples. 3. Known mass: place the configured mass at the center and keep it completely still. 4. Remove mass: the final fixed zero and recommended filter are verified before they are saved. The recommendation never overwrites the active preset selected below. There is no automatic zero tracking, Kalman adaptation or drift correction; later zero movement remains visible and measurable.

Measurement filter diagnostics

Select a fixed test preset and compare calibrated-unfiltered and filtered force from the same samples. The selection is stored independently on each board and remains active when you return to Scale or Realtime. The unity-DC-gain filter suppresses fast analog noise but deliberately leaves slow baseline drift visible.
